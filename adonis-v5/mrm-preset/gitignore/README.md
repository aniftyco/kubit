### Gitignore template

Creates `.gitignore` file in the root of your project. Following files and folders are ignored by default. However, you can add more to the template.

```
node_modules
coverage
test/__app
.DS_STORE
.nyc_output
.idea
.vscode/
*.sublime-project
*.sublime-workspace
*.log
build
docs
dist
shrinkwrap.yaml
```
